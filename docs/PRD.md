# 产品需求说明书 (MVP)

## 产品名称 (暂定)
**Examen Civique Prep**
(法国公民考试多语种备考平台)

---

## 1. 项目背景与目标

### 1.1 项目背景
自 2026 年起，法国对外国人申请以下身份引入统一的 **Examen Civique** (公民知识考试) 要求：
- 多年居留卡 (2年 / 4年)
- 十年长期居留卡
- 法国国籍 (入籍)

该考试以 **法语 QCM (选择题)** 形式进行，内容涵盖：
- 法国共和国价值观
- 国家制度与公共机构
- 公民权利与义务
- 法国历史、文化与社会生活

**目前存在的问题：**
- 官方资料 PDF 高度概括、缺乏练习。
- 缺少系统化题库与模拟考试。
- 对非法语母语者极不友好。
- 不同考试类型 (多年卡/十年卡/入籍) 难度差异大，但无分层学习工具。

### 1.2 用户痛点分析
目标用户主要为在法外国人，普遍存在以下痛点：
1.  **语言障碍**: 考试必须用法语，但大量考生法语水平有限 (A2–B1)。
2.  **缺乏针对性**: 不清楚自己对应哪种考试难度，无法判断哪些内容是重点。
3.  **备考效率低**: 只能反复阅读 PDF，没有练习，不知道是否“真的会了”。
4.  **时间碎片化**: 多为在职人群，需要随时随地刷题学习。

### 1.3 产品目标
本项目旨在打造一个：**以 Web / PWA 为核心、支持多语种解释的 Examen Civique 备考平台**。

**具体目标：**
- ✅ 提供结构化题库 + 模拟考试
- ✅ 按 多年卡 / 十年卡 / 入籍 分级
- ✅ 所有考试内容严格使用法语
- ✅ 提供多语种解释辅助理解 (中文、英语、阿拉伯语、印地语等)
- ✅ 支持手机端高频、碎片化使用 (PWA)
- ✅ 通过订阅或题库解锁实现商业化

---

## 2. 产品功能范围 (MVP 版本)

### 2.1 产品形态
- **平台类型**: Web 应用 (支持 PWA)
- **使用设备**: 手机 (优先)、电脑、平板
- **特点**: 无需应用商店下载，可添加至主屏幕使用

### 2.2 用户分组与考试类型
用户首次使用需选择目标考试类型之一：
1.  **多年居留卡** (2 / 4 年)
2.  **十年居留卡**
3.  **法国入籍**

系统将基于选择结果：
- 推荐对应难度题目
- 限定模拟考试难度

### 2.3 MVP 核心功能模块

#### 2.3.1 账号与基础设置
- 支持邮箱注册 / 登录
- 用户可在设置中选择：
    - 界面语言
    - 解释语言
    - 目标考试类型
- *📌 MVP 阶段不强制社交登录*

#### 2.3.2 多语种支持系统 (核心差异化)
**设计原则:**
- ✅ 所有题目与选项 **仅使用法语**
- ✅ 多语种仅用于「解释 / 解析」
- ❌ **不允许**在模拟考试中显示翻译

**支持语言 (首期):**
- 法语 (默认)
- 英语
- 中文
- 阿拉伯语
- 印地语

**适用范围:** 题目解析、概念解释、页面 UI 文本

#### 2.3.3 题库练习模块 (核心)
- **功能点:**
    - 按主题分类练习
    - 按考试类型筛选题目
    - 单题练习模式
    - 即时判题 + 法语解析
    - 可切换多语种解释
- **用户操作:** 收藏题目、标记不懂、查看错题记录

#### 2.3.4 模拟考试模块 (核心)
- **规则设定:**
    - 题量: 40 题
    - 模式: 计时模式
    - 通过线: 80%
- **限制:** 模拟考试期间 **仅显示法语**，禁止多语种解释，禁止查看解析。
- **考试结束后:** 显示得分、显示错题解析、提供复习建议。

#### 2.3.5 学习进度与反馈
- 学习进度统计
- 各主题正确率
- 模拟考试历史记录
- *📌 MVP 阶段不包含社交 / 排行榜*

### 2.4 MVP 明确不包含的功能 (产品边界)
- ❌ 社区 / 论坛
- ❌ 实时聊天
- ❌ 老师一对一辅导
- ❌ 视频课程
- ❌ AI 自动出题 (后期)
